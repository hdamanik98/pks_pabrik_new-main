Sistem Manajemen Pabrik Kelapa Sawit Lengkap

Odoo 17.0 LGPL-3 v1.0.0

🚛 Timbangan Digital

Sistem timbangan dengan state machine lengkap. Mendukung RFID, dual weighing (masuk/keluar), dan perhitungan netto otomatis.

👥 Portal Supplier

Supplier dapat mengakses portal untuk melihat history pengiriman, statistik, dan status verifikasi.

📊 Quality Control

Analisa kualitas TBS dengan 6 parameter. Grading otomatis (A-E) dengan perhitungan potongan berat.

📈 LHP & OER/KER

Laporan Harian Pabrik dengan perhitungan OER (Oil Extraction Rate) dan KER (Kernel Extraction Rate).

🔌 REST API

API lengkap dengan authentication untuk integrasi dengan sistem eksternal dan IoT devices.

📱 Kiosk Mobile

Interface kiosk timbangan yang mobile-responsive dengan OWL component.
